# Mental Health Trends Among Students (June–November)

This repository presents an **interactive data visualization project** that explores student mental health trends from **June to November**.  
The project highlights patterns in **stress, sleep, mood, and productivity**, using an embedded **Tableau Public dashboard** and a clean, responsive **HTML/CSS front end**.

## What The Project Does

- Visualizes key mental health indicators among students  
- Shows trends over time (June through November)  
- Helps identify patterns in stress, sleep, mood, and productivity  
- Provides an interactive and user-friendly data dashboard

## Repository Contents

- `index.html` – Main interactive webpage embedding the dashboard  
- `styles.css` – Custom styles for the webpage layout  
- `Mental_Health_Trends_Dashboard.pdf` – Downloadable project report  
- Other supporting files

## How to Run

1. Clone this repository  
2. Open `index.html` in a browser  
3. Interact with the embedded Tableau dashboard

> No server setup required — it runs purely in the browser.
